# Scuffed AI
A local ai

## How to setup:
1. Clone this repo
2. Create a virtual environment
3. Install the necessarry dependencies from requirements.txt with the command `pip install -r requirements.txt`
4. Run app.py
5. Go to the link outputted in the terminal
6. Enjoy Scuffed AI
